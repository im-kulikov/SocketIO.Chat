# SocketIO.Chat #

A simple implementation of the chat using the NodeJS, SocketIO, ExpressJS

### Technologies: ###

* NodeJS
* SocketIO
* Jade template engine
* Sass
* CoffeeScript
* Redis for store

### How do I get set up? ###

* $> git clone <repo-url>
* $> npm install
* $> npm start
