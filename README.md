# SocketIO.Chat #

Simple realisation of chat with NodeJS + SocketIO + Express

### Technologies: ###

* NodeJS
* SocketIO
* Jade template engine
* Sass
* CoffeeScript
* Redis for store

### How do I get set up? ###

* $> git clone <repo-url>
* $> npm install
* $> npm start
